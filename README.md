# my-site

A personal portfolio website built with Next.js, React 19, TypeScript, and Tailwind CSS v4. Project may update and continue to change over time.

## Stack

- **Framework:** Next.js 16
- **UI:** React 19, Tailwind CSS v4
- **Language:** TypeScript
- **Icons:** Tabler Icons
- **Map:** Leaflet

## Features

- Grid dashboard with location map, status cards, and Snapple facts
- Featured projects section with screenshots and tech stack
- Theming system using Catppuccin palettes with 14 accent color options

## Getting Started

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:3000`.
